# VibeChef

> AI-powered recipe generator - tell it your ingredients, pick a vibe, and let Gemini cook.

## Features

- **AI Recipe Generation** - Powered by Google Gemini 2.5 Flash
- **Voice Input** - Dictate your ingredients hands-free via Speech-to-Text
- **Camera Ingredient Detection** - Snap photos and let Gemini identify ingredients
- **Dietary Filters** - Vegetarian, Gluten-Free, Spicy
- **Recipe History** - Save and browse your generated recipes (Firebase Firestore)
- **Authentication** - Email/password sign-in/sign-up with Firebase Auth
- **Dark Mode** - Toggle between light and dark themes
- **Share and Copy** - Share recipes via any app or copy to clipboard

## Tech Stack

Kotlin, Jetpack Compose, Material 3, Google Gemini 2.5 Flash, Firebase Auth, Cloud Firestore, Navigation Compose, Gradle Kotlin DSL

## Setup

1. Clone the repo
2. Set your Gemini API key as env variable GEMINI_KEY
3. Open in Android Studio and sync Gradle
4. Run on device/emulator (min SDK 28)
